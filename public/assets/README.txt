INSTRUCTIONS FOR ADDING IMAGES
==============================

This folder (`public/assets`) is where you can place images for the general website design.

1. Website Background
   - File name: `site-background.jpg`
   - Purpose: This image will appear as the background for the entire website.
   - To change it: Delete the existing file and paste your own image here, renaming it to `site-background.jpg`.

2. Home Page Hero Images
   - File name: `hero-dal.jpg` (The top-left image in the hero section)
   - File name: `hero-sweets.jpg` (The bottom-right image in the hero section)
   - To change them: Replace these files with your own images, keeping the same names.

3. Other Assets
   - You can add other images here and reference them in your code as `/assets/filename.jpg`.
